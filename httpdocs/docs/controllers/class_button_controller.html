class ButtonController¶

Class manages Pushbuttons After construct needs to be configured with Informations about which Pin to Listen and where to send. SendsMidi Events on getData triggered if status changed

Public Functions

ButtonController()¶

Buttoncontroller constructor

Based on pinlist a debounced Array will be created in button

Parameters

  • amount: amount of Buttons the controller should be able to handle

  • pinList: List of pin numbers matches Button by Button e.g. {1, 2, 3}

  • midiValueList: List of Midivalues matches Button by Button e.g. {21, 22, 23} Hint on Buttonmapping: First Entry of pinList will be matched to first entry of midiValueList etc.
